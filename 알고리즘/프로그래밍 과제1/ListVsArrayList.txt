List vs ArrayList

List<Integer> list = new ArrayList<Integer>();
ArrayList<Integer> list = new ArrayList<Integer>();

List
인터페이스
list를 List 자료형으로 선언한 경우, ArrayList, LinkedList로 구현할 수 있음


ArrayList
클래스
Public class ArrayList<E> extends AbstractList<E> implementation List<E>, RandomAcess, Cloneable, java.io.Serializable
List가 제공하는 기능들을 다 제공할 수 있음

ArrayList 자료형인 경우 LinkedList로 새롭게 구현할 수 없음
